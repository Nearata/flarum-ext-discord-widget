# Discord Widget

## Install

```sh
composer require nearata/flarum-ext-discord-widget:"*"
```

## Update

```sh
composer update nearata/flarum-ext-discord-widget:"*"
php flarum migrate
php flarum cache:clear
```

## Remove

```sh
composer remove nearata/flarum-ext-discord-widget
php flarum cache:clear
```
